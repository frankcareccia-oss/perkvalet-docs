# PerkValet Documentation

This directory is the canonical documentation root for the PerkValet project.

## Structure

### architecture
Core platform architecture, system maps, role model, actor model, environment model.

### ui
Admin and merchant UI contracts, staffing/UI behavior, style guide, support footer spec.

### security
Authorization contracts, capability registry, roles/capabilities architecture.

### billing
Billing specifications, invoice flows, guest pay flow, and billing test assets.

### threads
Thread plans, handoffs, frozen status docs, and templates.

### qa
QA test plans and execution notes.

### archive
Deprecated or superseded documentation kept for reference only.
